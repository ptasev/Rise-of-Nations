Installation

To install, simply EXTRACT everything from the zip file to a desired location.

Application Information

This application converts various files to common file formats and back for the Rise of Nations game.

Usage

Drag and drop (or use Open With) any supported file type on the executable to convert it to a common file format and back.

Pass the -h option on the command line to get help text for any command.

Tips for bh3/bha and glTF conversion:

  • The format supports only one bone assigned to a vertex (no blending).
  • The format does not support bone scaling.
  • The format supports only one material/texture.
  • Only skinned meshes will be converted.
  • Animations (bha) can only be applied to models (bh3) with the same exact skeleton hierarchy.
  • When animating, it is best to use 30 FPS.
  • Attack animation frame count must be double the recharge number in the unitrules.xml file.
